# Custos Handbook

A plain-language manual for Custos. Read this to understand exactly how every feature works and to explain it to anyone. Mostly plain English, light technical detail where it helps. Updated one feature at a time as Blocks close.

## What Custos is

Custos is a private AI assistant that answers questions from a business's own documents. It cites real sources, refuses to answer when it does not know, and is hardened against prompt injection, data leakage, and unauthorized access. It is built for businesses that need "an AI that knows our stuff" but will not trust one that is not secure.
